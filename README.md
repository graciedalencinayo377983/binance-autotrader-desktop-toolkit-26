## 🧠 Overview

**BinanceTrader Toolkit** is a heavily optimized, asynchronous C++ toolkit specifically designed to monitor exchange-side market data, streamline order workflows, and synchronize portfolio views from a standalone command-line environment.

Whether you are tracking price movement, testing automation pipelines, or exporting account-side market snapshots for analysis, BinanceTrader Toolkit bridges the gap between exchange APIs and your local workstation quickly and natively.

## 🚀 Features

*   **⚡ Hyper-Fast Market I/O:** Built around concurrent websocket and REST handling for responsive market data collection.
*   **🔓 Standalone Architecture:** Runs independently without requiring a browser trading terminal to remain open.
*   **📦 Deep Exchange Data Access:** Processes ticker streams, order status snapshots, portfolio balances, and symbol metadata directly.
*   **🔗 Workflow-Oriented Automation:** Connects account-side events to logging pipelines, strategy testing tools, and local scripts.
*   **🎨 Cross-Ecosystem Support:** Output integrates with dashboards, analytics wrappers, and archive-oriented trading utilities.


## 🧩 Tech Stack

*   **Core:** Modern C++20
*   **Network:** REST / WebSocket API clients
*   **Parsing:** nlohmann/json
*   **FileSystem:** Native `<filesystem>` API
*   **Analytics:** TA-oriented indicator integration
